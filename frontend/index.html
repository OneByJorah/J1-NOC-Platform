📘 Setup Guide

To expose logs or status as HTTP:

• Windows: Add static file serving to DC_ReplAPI.ps1 or create a simple IIS site pointing to C:\DC_Dashboard\

• Ubuntu: Run python3 -m http.server 8000 in the log directory, or use nginx/apache

• NTP Status JSON: Run NTP_Monitor.ps1 (included below) which exports ntp_status.json to C:\DC_Dashboard\

• Ubuntu Status JSON: Run ubuntu_status.sh as a cron job every 1 minute — exposes /var/www/html/status.json

PowerShell for NTP_Monitor.ps1:

# NTP_Monitor.ps1 — run every 5 minutes via Task Scheduler
# Queries ntpq -p and exports ntp_status.json

$output = @{ RunAt = (Get-Date -Format "yyyy-MM-dd HH:mm:ss"); Clients = @() }
try {
  $ntpq = ntpq -p -n 2>$null
  foreach ($line in $ntpq | Select-Object -Skip 2) {
    if ($line -match '^[\*\+\-ox\s](\S+)\s+\S+\s+\d+\s+\d+\s+(\d+)\s+([\d\.]+)\s+([\d\.\-]+)\s+([\d\.]+)') {
      $output.Clients += @{
        Address  = $matches[1]; Reach = $matches[2]
        Interval = $matches[3]; Offset = [float]$matches[4]
        Jitter   = [float]$matches[5]
        Status   = if([math]::Abs([float]$matches[4]) -lt 128){'ok'}elseif([math]::Abs([float]$matches[4]) -lt 512){'warn'}else{'crit'}
        LastSeen = (Get-Date -Format "HH:mm:ss")
      }
    }
  }
} catch {}
$output | ConvertTo-Json -Depth 3 | Out-File "C:\DC_Dashboard\ntp_status.json" -Encoding UTF8
