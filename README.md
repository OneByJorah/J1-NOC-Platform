# J1 NOC Platform (JNOP)

Enterprise Network Operations Center Platform

## Features
- Overview Dashboard
- Network Monitoring (Ping/SNMP)
- LDAP / AD Replication
- PBX Monitoring
- Chrony NTP Dashboard
- osTicket Integration
- DNS Resolver Tools
- Admin Panel
- SMTP / Telegram / Teams Notifications
- Historical Metrics
- Ollama AI Assistant

## Stack
- React Frontend
- FastAPI Backend
- PostgreSQL
- Redis
- Prometheus
- Grafana
- Docker Compose

## Security
All credentials use sample placeholders and must be configured before deployment.
